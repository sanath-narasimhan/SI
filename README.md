# SRG Vedanta School - Website Revamp

A premium, modern, and interactive digital platform for the SRG Vedanta School, specifically re-wired for **Corporate, School, and Institutional outreach**.

## 🚀 Key Features Implemented

### 🎨 Design & Aesthetic
- **Premium Glassmorphism**: High-fidelity hero banner with frosted glass effects, translucent edges, and high-blur backdrops.
- **Modern Typography**: Sophisticated font stack using *Playfair Display* for headings and *Inter* for body text.
- **Unified Branding**: Exact textual alignment with official school messaging and branding.

### 🔱 Immersive Visuals
- **Kundalini Trisula Animation**: A high-density spiraling energy system (Ida and Pingala) that weaves around a central axis (Sushumna) based on scroll progress.
- **Rising Prana Tip**: A dynamic, luminous focal point that tracks the spiritual ascent.
- **Fixed Meditation Silhouette**: A subtle, pulsing background silhouette that anchors the visual narrative.

### 🧩 Interactivity & Layout
- **3D Program Outreach Cards**: Interactive 2x2 grid with flip-animations for different institutional audiences.
- **Aligned 5-Stage Roadmap**: A transparent, expansive journey section perfectly synchronized with the background energy visuals.
- **Impact Section**: A dual-column layout featuring core philosophy and interactive "Proven Benefits" cards.
- **Light & Dark Balance**: A clean "Modern Spiritual" light theme with a solid, professional dark "Contact Us" anchor.

## 🛠️ How to Run Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sanath-narasimhan/SI.git
   cd SI
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   *The site will be available at `http://localhost:5173/`*
